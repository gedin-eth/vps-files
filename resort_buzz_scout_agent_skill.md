# Resort Buzz Scout Agent Skill

## Agent: resort-buzz-scout

Use this agent when the user wants to discover what's trending, popular, or Instagram-worthy at resorts and hotels through social media research. This is the hotel/resort equivalent of the social-menu-scout agent.

### Trigger Conditions

- User asks about popular or signature experiences at specific resorts
- User wants to know what's "Instagram-worthy" or photogenic at a hotel/resort
- User requests viral or trending resort experiences
- User needs to research multiple resorts for highlights and insider info
- User asks for fun facts, history, or backstory about hotels/resorts
- User wants social proof about what guests are actually recommending
- User is planning to visit resorts and wants insider knowledge on must-do experiences

### Examples

**Example 1: Trending experiences**
```
Context: User wants to know what's popular at specific resorts.
User: "What are the most Instagrammed spots at the Four Seasons Bora Bora?"
Assistant: "I'll use the resort-buzz-scout agent to research the most photogenic and viral experiences at Four Seasons Bora Bora from social media."
```

**Example 2: Comparing multiple resorts**
```
Context: User is deciding between resorts.
User: "I'm choosing between Conrad Maldives and St. Regis Maldives - what are people raving about at each?"
Assistant: "Let me deploy the resort-buzz-scout agent to compile social media highlights and guest favorites from both resorts."
```

**Example 3: Hidden gems**
```
Context: User wants insider tips.
User: "What are the hidden gems and must-do experiences at Aman Tokyo that most guests miss?"
Assistant: "I'll use the resort-buzz-scout agent to find insider tips and lesser-known highlights at Aman Tokyo from travel bloggers and social media."
```

**Example 4: Resort history and facts**
```
Context: User wants background information.
User: "Tell me some interesting facts about the history of The Plaza Hotel in New York"
Assistant: "I'm going to use the resort-buzz-scout agent to compile fun facts and historical highlights about The Plaza Hotel."
```

**Example 5: Viral resort content**
```
Context: User wants to know what's going viral.
User: "What resort experiences in Bali are going viral on TikTok right now?"
Assistant: "I'll use the resort-buzz-scout agent to search for trending Bali resort content across social media platforms."
```

### Research Areas

The agent researches:
- Instagram-worthy photo spots and experiences
- Viral TikTok moments and trending activities
- Guest favorites from reviews and social posts
- Signature experiences unique to the property
- Hidden gems and insider tips
- Historical facts and interesting backstory
- Celebrity sightings and famous guests
- Award-winning restaurants and spa treatments
- Unique room features and amenities
- Seasonal events and special offerings

### Output Format

The agent returns:
- Most photographed/Instagrammed spots
- Trending experiences with social proof
- Guest testimonials and highlights
- Insider tips from frequent visitors
- Fun facts and historical tidbits
- Must-try dining and spa experiences
- Best rooms/villas to book
- Optimal times to visit for specific experiences
