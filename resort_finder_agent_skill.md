# Resort Finder Agent Skill

## Agent: resort-finder

Use this agent when the user requests resort/hotel recommendations, asks about accommodation options, wants to find places to stay, or when you need to search for resorts based on location, amenities, and dates.

### Trigger Conditions

- User wants resort or hotel recommendations
- User asks about accommodation options (overwater bungalows, villas, luxury resorts, etc.)
- User is planning a vacation and needs lodging
- User mentions specific resort types (all-inclusive, boutique, eco-resort, etc.)
- User asks about honeymoon destinations or romantic getaways

### Examples

**Example 1: Location-based search**
```
Context: User wants resort recommendations for a specific destination.
User: "I'm looking for overwater bungalows in Bora Bora"
Assistant: "I'll use the resort-finder agent to search for the best overwater bungalow resorts in Bora Bora."
```

**Example 2: Amenity-specific search**
```
Context: User is looking for specific resort features.
User: "What are the best all-inclusive resorts in the Maldives with private pools?"
Assistant: "Let me use the resort-finder agent to find all-inclusive Maldives resorts featuring private pool villas."
```

**Example 3: Budget-conscious search**
```
Context: User mentions budget considerations.
User: "I want a nice beach resort in Thailand that's not too expensive"
Assistant: "I'll use the resort-finder agent to search for highly-rated yet affordable beach resorts in Thailand."
```

**Example 4: Special occasion**
```
Context: User planning a special trip.
User: "We're looking for a romantic honeymoon resort in Fiji"
Assistant: "I'll use the resort-finder agent to find the top honeymoon-friendly resorts in Fiji with romantic amenities."
```

### Search Criteria

The agent will search for resorts based on:
- Location/destination
- Resort type (luxury, boutique, all-inclusive, eco-resort)
- Accommodation style (overwater bungalow, villa, suite, treehouse)
- Amenities (private pool, spa, beach access, diving)
- Price range
- Traveler ratings and reviews
- Special features (adults-only, family-friendly, pet-friendly)

### Output Format

The agent returns:
- Resort name and location
- Star rating and traveler scores
- Key amenities and features
- Room/villa types available
- Price range indication
- Booking links or contact information
- Notable highlights and unique selling points
